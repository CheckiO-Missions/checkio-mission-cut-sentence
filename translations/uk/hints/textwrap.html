Чи можете ви дати мені ідею, з чого можна почати?

Звичайно. Стандартна бібліотека Python Textwrap - це саме те, що вам потрібно для таких завдань. Будь ласка, вивчіть опис і спробуйте знайти необхідні методи.

Ви впевнені, що якийсь з цих методів допоможе мені розв'язати задачу?

Так... Перевір метод shorten().

Чомусь функція не допомагає.

Це тому, що функція сама по собі не є рішенням. Спочатку ви повинні використати її, щоб вирізати лінію. 

import textwrap

shortened = textwrap.shorten(original, width, placeholder='', break_long_words=False)

Потім ви повинні додати ... в кінці. 

return shortened + '...'

І не забувайте, що ви повинні повернути оригінальне речення, якщо воно досить коротке.

Чудово! Чи можете ви допомогти мені зібрати все це разом в одне рішення.

Звичайно. Так можна вирішити місію. 

import textwrap

def cut_sentence(line: str, length: int) -> str:

    if len(line) <= length:
        return line
    else:
        return textwrap.shorten(line, width=length,
        placeholder="", break_long_words=False) + "..."
