Підкажи, з чого мені розпочати?

Ти можеш завжди почитати Python Стандартні Бібліотеки.

Можна конкретніше?

Звичайно. Textwrap тобі допоможе.

Ти впевнений, що якась з тих методів мені допоможе?

Так, почитай про метод shorten().

Чомусь вона не працює як треба...

Щож, це тому що функція сама по собі не є рішенням. Спочатку використай її, щоб розділити рядок: 

import textwrap


shortened = textwrap.shorten(original, width, placeholder='', break_long_words=False)

Потім додай ... в кінець: 

return shortened + '...'

І не забувай, ти маєш повернути вхідний рядок, якщо він досить коротке.

Чудово! Можеш допомогти мені це все об'єднати в одне рішення?

Звичайно. Ось як може бути вирішена місія. 

import textwrap


def cut_sentence(line: str, length: int) -> str:

    if len(line) <= length:
        return line
    else:
        return textwrap.shorten(line, width=length,
        placeholder="", break_long_words=False) + "..."
