Pouvez-vous me donner une idée de la façon dont je peux commencer?

Bien sûr, la bibliothèque standard de Python Textwrap est exactement ce dont vous avez besoin pour ce genre de tâches. Veuillez consulter la description et essayer de trouver les méthodes nécessaires.

Êtes-vous sûr que l'une de ces méthodes peut m'aider à résoudre la mission?

Oui. Vérifiez la méthode shorten().

Pour une raison quelconque, la fonction ne vous aide pas.

C'est parce que la fonction elle-même n'est pas une solution. Vous devez d'abord l'utiliser pour couper la ligne. 

import textwrap

shortened = textwrap.shorten(original, width, placeholder='', break_long_words=False)

Ensuite, vous devez ajouter ... à la fin. 

return shortened + '...'

Et n'oubliez pas que vous pouvez renvoyer une phrase originale si elle est suffisamment courte.

Génial! Pouvez-vous m'aider à rassembler tout cela dans une solution?

Bien sûr. Voici comment la mission peut être résolue. 

import textwrap

def cut_sentence(line: str, length: int) -> str:

    if len(line) <= length:
        return line
    else:
        return textwrap.shorten(line, width=length,
        placeholder="", break_long_words=False) + "..."
