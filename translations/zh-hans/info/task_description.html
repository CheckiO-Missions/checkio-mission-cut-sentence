你在这个任务中的任务是将一个句子截断，使其长度不超过指定的字符数。

如果给定的句子已经足够短，你不必对它做什么。但如果需要截断，则必须在缩短后的句子末尾连接一个省略号（"..."）来表示省略。

缩短的句子可以包含完整的单词和空格。
它既不能包含截断的单词，也不能包含尾部的空格。
省略号已被计入允许的字符数，因此它不计入给定的长度。

输入：两个参数：

  • 作为一个字符串的单行句子 (str{% if is_js %}ing{% endif %});
  • 被截断的句子的最大长度为一个整数 {% if is_js %}(number){% else %}(int){% endif %}.

输出：缩短的句子加上省略号（如果需要），作为一个 单行字符串 (str{% if is_js %}ing{% endif %}).

{{init_code_tmpl}}

前提条件：

  • line.startingwith(' ') == False
  • 0 < len(line) ≤ 79
  • 0 < 长度 ≤ 76
  • all(char in string.ascii_letters + '' for char in line)
