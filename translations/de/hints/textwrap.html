Aus irgendeinem Grund hilft die Funktion nicht weiter.

Nun, das liegt daran, dass die Funktion selbst keine Lösung ist. Zuerst sollten Sie die Zeile mit ihr abschneiden. 

import textwrap

shortened = textwrap.shorten(original, width, placeholder='', break_long_words=False)

Dann sollten Sie am Ende ... hinzufügen. 

return shortened + '...'

Und vergessen Sie nicht, dass Sie einen Originalsatz zurückgeben sollten, wenn er kurz genug ist.

import textwrap

def cut_sentence(line: str, length: int) -> str:

    if len(line) <= length:
        return line
    else:
        return textwrap.shorten(line, width=length,
        placeholder="", break_long_words=False) + "..."
