Ihre Aufgabe in dieser Mission ist es, einen Satz auf eine Länge zu kürzen, die eine bestimmte Anzahl von Zeichen nicht überschreitet.

Wenn der vorgegebene Satz bereits kurz genug ist, müssen Sie nichts mehr daran ändern. Wenn er jedoch gekürzt werden muss, müssen Sie dies durch Anhängen einer Ellipse ("...") an das Ende des gekürzten Satzes anzeigen.

Der gekürzte Satz kann vollständige Wörter und Leerzeichen enthalten.
Er darf weder abgeschnittene Wörter noch nachgestellte Leerzeichen enthalten.
Die Ellipse wurde bei der zulässigen Anzahl von Zeichen berücksichtigt und wird daher nicht auf die angegebene Länge angerechnet.

Eingabe: Zwei Argumente:

  • einzeiliger Satz als Zeichenkette (str{% if is_js %}ing{% endif %});
  • maximale Länge des gekürzten Satzes als Ganzzahl {% if is_js %}(number){% else %}(int){% endif %}.

Ausgabe: Der gekürzte Satz plus die Ellipse (falls erforderlich) als ein einzeilige Zeichenfolge (str{% if is_js %}ing{% endif %}).
