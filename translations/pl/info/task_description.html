Państwa zadaniem w tej misji jest skrócenie zdania do długości nieprzekraczającej określonej liczby znaków.

Jeśli podane zdanie jest już wystarczająco krótkie, nie trzeba nic z nim robić. Jeśli jednak trzeba je skrócić, pominięcie należy zaznaczyć, łącząc wielokropek ("...") na końcu skróconego zdania.

Skrócone zdanie może zawierać pełne słowa i spacje.
Nie może zawierać obciętych słów ani spacji.
Elipsa została uwzględniona w dozwolonej liczbie znaków, więc nie wlicza się do podanej długości.

Wejście: Dwa argumenty:

  • zdanie jednowierszowe jako ciąg znaków (str{% if is_js %}ing{% endif %});
  • maksymalna długość obciętego zdania jako liczba całkowita {% if is_js %}(number){% else %}(int){% endif %}.

Dane wyjściowe: Skrócone zdanie plus wielokropek (jeśli jest wymagany) jako jednowierszowy ciąg znaków (str{% if is_js %}ing{% endif %}).
